## Getting Started

### Built With

- [![Vue][vue.js]][vue-url]
<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Prerequisites

These are the only requisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Levifernandes00/daylight.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Execute program
   ```sh
   npm run serve
   ```
